# Trekbag

A minimalist packing list application to help you stay organized for your trips.

## Features

- **Add Items**: Easily add items to your packing list.
- **Track Progress**: Mark items as packed or unpacked to keep track of your progress.
- **Remove Items**: Delete items you no longer need.
- **Sort & Filter**: Organize your list to see what's most important.
- **Persistent Storage**: Your list is saved automatically using local storage (via Zustand).

## Tech Stack

- **React**: UI library for building the interface.
- **TypeScript**: Strongly typed programming language that builds on JavaScript.
- **Vite**: Fast build tool and development server.
- **Zustand**: Small, fast, and scalable bearbones state-management solution.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```
